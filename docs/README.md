# Project Documentation

This folder contains supporting documentation for the **Student Task Tracker** project.

## Included Documents
- **team_project_plan.md** – outlines the project concept, goals, responsibilities, and timeline
- **team_project_pseudocode.md** – contains starter pseudocode for the main project features
- **test_plan.md** – describes how the project will be tested
- **test_results.md** – used to record testing outcomes during implementation
- **student_task_tracker_UML_diagram.png** – UML diagram for the project design

## Purpose
These documents support the design, planning, implementation, and testing phases of the Student Task Tracker project.
