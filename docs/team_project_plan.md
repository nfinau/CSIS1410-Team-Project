## Project Description
The Student Task Tracker is an application designed to help students organize and manage their academic responsibilities. Users can create tasks by entering a title, description, and priority level. They can also edit tasks, delete tasks, mark tasks as complete, and filter tasks by priority. The purpose of the project is to provide a simple and effective way for students to stay organized and manage their workload more efficiently.

## Main Features
- Add a new task
- Edit an existing task
- Delete a task
- Mark a task as complete
- Filter tasks by priority
- Store and manage multiple tasks

## Planned Classes
- **Task** – stores task details such as title, description, priority, completion status, and task ID
- **TaskManager** – manages task creation, deletion, updating, filtering, and storage
- **GuiTaskTracker** – handles user interaction
- **TaskPriority** – enum for LOW, MEDIUM, and HIGH priorities
